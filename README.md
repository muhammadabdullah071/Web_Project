# Web Project

This repository contains a Laravel web application located in the `store_backend/` directory.

## Setup

1. Copy `.env.example` to `.env`
2. Install PHP dependencies with `composer install`
3. Install frontend dependencies with `npm install` or `yarn`
4. Generate app key with `php artisan key:generate`
5. Run migrations with `php artisan migrate`
6. Start the development server with `php artisan serve`

## Notes

- `store_backend/` contains the Laravel application source code.
- `.gitignore` is configured for Laravel and common PHP/node build artifacts.
